The app consists of several components, each serving a specific visualization purpose:

    App.js: This is the main entry point of the application. It provides a file input field for users to upload JSON data. Once data is uploaded, it's used to populate various visualization components.
    ParallelChart.js: This component renders a parallel coordinates chart, allowing users to compare and analyze data across multiple dimensions. The chart's lines can be selected, highlighting them in red for emphasis. Data lines are clustered together based on their values.
    TreeView.js: This component renders a hierarchical tree visualization. Each node in the tree corresponds to a data point. Users can click on nodes to select them, changing their color to red for emphasis. Additionally, a selected node is associated with a line in the ParallelChart.
    ThreeJS.js: This component renders a 3D cube using the Three.js library. The cube rotates along its x and y axes, creating a simple 3D animation.
    ClusterScatterplot.js: This component renders a scatter plot with clustering. The data points are clustered into distinct groups, and a legend with different colors represents each cluster. Users can hover over data points to view additional information.
